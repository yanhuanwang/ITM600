# 1\. Introduction 

In this project, we aim to apply machine learning techniques to predict food truck profits based on city population. This project is based on the PopulationProfit dataset, which contains historical information about city population sizes and corresponding profits. The goal is to build a predictive model that can help decision-makers determine whether opening a food truck in a new city would be profitable. Machine learning offers a data-driven approach to solving such business problems. By analyzing patterns in historical data, we can build predictive models that generalize to unseen data. This project focuses on data preprocessing, which is an essential step before model training. Proper data preprocessing ensures high-quality input data and improves model performance.

# 2\. Dataset Description 

[https://github.com/yanhuanwang/ITM600](https://github.com/yanhuanwang/ITM600)   
The dataset used in this project is PopulationProfit.csv. This dataset contains two main features: 1\. Population of the city (in 10,000s) 2\. Profit of food truck (in $10,000s) Each row represents a city and the corresponding profit generated by a food truck. The dataset contains both positive and negative profit values. Negative values represent financial losses. The dataset is relatively small and simple, which makes it suitable for learning basic machine learning concepts. However, even simple datasets require careful preprocessing to ensure reliable results.

# 3\. Data Exploration 

Before applying machine learning algorithms, the dataset was explored using descriptive statistics and visualization techniques. A scatter plot was created to visualize the relationship between population and profit. The visualization shows a linear relationship between population size and profit. This indicates that linear regression is an appropriate model for this dataset. Understanding the relationship between variables helps guide model selection.

# 4\. Data Preprocessing 

Data preprocessing involves cleaning and preparing data for machine learning algorithms. The following preprocessing steps were applied: • Data Loading using NumPy • Reshaping input data • Feature selection • Adding intercept term • Data visualization The dataset did not contain missing values, so no imputation was necessary. Additionally, since the dataset contains only one feature, feature scaling was not required for this task.

# 5\. Planned Machine Learning 

Method Linear regression was selected as the machine learning method for this project. Linear regression models the relationship between independent variables and dependent variables using a linear equation. The model was trained using gradient descent to minimize the cost function. The cost function used in this project is Mean Squared Error (MSE). Gradient descent iteratively updates model parameters until convergence. This approach provides a simple and interpretable model for predicting profits.

# 6\. Expected Results 

The expected outcome of this project is a predictive model capable of estimating food truck profits based on population size. The model will also help identify cities with potential profitability. This project demonstrates how machine learning can support business decisions using historical data.
