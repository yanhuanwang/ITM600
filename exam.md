Examination: Capstone Project
Capstone Project: Apply acquired knowledge to solve a real-world problem using machine learning techniques.

Activities:

Students will work on a capstone project that applies machine learning techniques to a real-world problem using a chosen dataset.
They will present their findings and solutions either as an on-campus poster or through an online presentation, along with a detailed project report.
 

Final Project Report Submission Guidelines:

Submit your Final Project Report on time.
The report should be 4-5 pages long, written using the IEEE conference template provided on Canvas.
Important: We will be using plagiarism detection tools, so do NOT copy from each other or any other resources.

Report Writing Guidelines:

Create a GitHub repository for your code and provide a reference to your GitHub page in the report.
Your report must include the following sections:
Abstract (a summary of what, why, how, and what results are achieved).
Introduction (what)
Problem Statement (why)
Methodology (machine learning models chosen for the project) (how)
Results and Analysis (what results)
Conclusion
References
Presentation Guidelines: 20 minutes presentation and 10 minutes Q/A.

Prepare a formal presentation of your completed project. The presentation should follow a structure similar to your report, including the following:

Introduction: Introduce the project and the problem you are addressing.
Methodology: Discuss the approach you applied to solve the problem, including the machine learning models you used.
Results and Analysis: Present your results along with your analysis.
Conclusion: Summarize your key findings and final conclusions.
